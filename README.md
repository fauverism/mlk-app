# Bridges - Finding Unity in Difference

A Martin Luther King Jr. Day celebration app that helps people find common ground between seemingly opposing perspectives.

## Features

- **Dynamic AI-Powered Bridge Finding**: Uses Claude AI to discover shared values and common ground
- **Interactive Examples**: Pre-loaded example scenarios to explore
- **Auto-Save**: Automatically saves your work in the browser
- **Character Counters**: Real-time feedback on input length
- **Keyboard Shortcuts**: Ctrl/Cmd + Enter to explore bridges
- **Copy Functionality**: Easy sharing of discovered bridges
- **Beautiful UI**: Elegant sepia-toned design honoring MLK's legacy

## Getting Started

1. Get your Anthropic API key from [console.anthropic.com](https://console.anthropic.com/)
2. Enter your API key in the app
3. Enter two different perspectives
4. Click "Explore Bridges" or press Ctrl/Cmd + Enter
5. Discover the common ground!

## Technologies

- Vanilla JavaScript
- Claude AI API
- HTML5/CSS3
- LocalStorage for persistence
